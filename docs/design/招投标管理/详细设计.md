## 1. 模块概述

- **路由**: /wande/bidding
- **菜单层级**: 商务中心 > 招投标管理
- **权限标识**: wande:bidding:*
- **所属模块**: module:fullstack
- **Sprint**: Sprint-2（💰 商务全闭环）

### 1.1 业务定位

从项目矿场发现商机到中标签约的全链条管理，覆盖：资质证书管理、检测报告管理、投标立项与进度、投标文件编制、报价决策与保证金。

### 1.2 数据流

```
矿场(Sprint-1) → 投标申请(审批) → 投标立项(C)
  → 标书申请(审批) → 资质匹配(A) + 检测报告(B) → 投标文件编制(D)
  → 授权申请(审批,按需)
  → 报价决策(E) → 开标/中标 → 执行管理(Sprint-2)接手
```
