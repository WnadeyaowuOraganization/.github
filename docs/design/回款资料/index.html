需求说明 回款资料管理总览
▼
📌 页面目的 体游行业全过程资料管理主入口，覆盖从投标报名到交付验收的5阶段资料Checklist，降低因资料缺失/延误导致的回款风险。含全阶段看板、甲方表单填写辅助、甲方资料需求模板库3个Tab

⚡ 关键操作 Tab切换 / 按阶段筛选资料 / 批量上传 / 一键填充甲方表单字段 / 下载资料需求模板 / 触发催收提醒
