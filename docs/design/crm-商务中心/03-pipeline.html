需求说明 商机管道
▼
📌 页面目的 以列表和看板双视图统览全部商机流转状态，支持7阶段（N→C→B→S→P→A→D）可视化管理，解决明道云手动改状态的痛点。

🎯 核心功能 列表/看板视图切换 / 7阶段Kanban列（每列显示商机卡片） / 多条件筛选（阶段/负责人/业务线/时间） / 15种来源分类标注
⚡ 关键操作 切换列表/看板视图 / 点击[推进阶段]更新商机至下一阶段 / 看板拖拽推进（Sprint-2） / 点击商机名进入详情
