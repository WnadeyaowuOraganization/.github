需求说明 定时任务管理
▼
📌 页面目的 统一查看平台所有Cron定时任务执行情况，支持手动触发和紧急暂停，解决定时任务静默失败不可见的问题
👤 主要用户 吴耀（超管）, 伟平（开发者）
🎯 核心功能 定时任务列表（Cron表达式/上次执行/状态） / 手动触发弹窗 / 执行日志抽屉 / 暂停/启用操作
⚡ 关键操作 手动触发指定Cron任务 / 暂停/启用任务 / 查看最近N次执行日志
✅ 验收要点 任务列表与CronTaskManagementController一致 / 手动触发后日志可见 / 暂停操作实时生效
📖 详细设计文档：超管驾驶舱 详细设计.md
