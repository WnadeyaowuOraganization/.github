📋 招投标全链条管理 — 原型说明

Tab页 文件名 优先级 功能描述
投标总览 tab-overview.html P0 全局看板：在投项目状态分布、本月KPI、里程碑倒计时、待办清单、保证金占用概览
资质证书管理 tab-certificates.html P0 证书数字档案库（6分类14字段）、3级预警（90/30/到期）、匹配引擎、续期进度
检测报告管理 tab-reports.html P0 6类报告归档、有效性评估、覆盖分析（TSG 71-2023）、产品关联、计划管理
投标立项与进度 tab-projects.html P1 Go/No-Go决策引擎、项目看板（6列）、任务自动分配、里程碑倒计时、审批流（投标申请/标书申请/授权申请）
投标文件编制 tab-documents.html P1 模板库、招标文件AI解析、多人协作编制、合规检查引擎、废标风险扫描、偏离表
报价决策与保证金 tab-pricing.html P2 成本模板、竞争对手报价库、基准价模拟器、保证金台账（投标/履约/质保3类）、资金占用分析
跨境市场准入 tab-international.html P3 4子区块：采购联盟资格(§2.28) + 跨境资格评估向导(§2.29) + 市场流程知识库(§2.30) + 国际项目数据(§2.31)

数据流

矿场(Sprint-1) → 投标申请(审批) → 投标立项(C) → 标书申请(审批) → 资质匹配(A) + 检测报告(B) → 投标文件编制(D) → 授权申请(审批,按需) → 报价决策(E) → 开标/中标 → 执行管理接手

3个审批流程

审批流程 触发时机 审批级数 所在Tab
投标申请 矿场确认商机后，决定"要不要投" 商务→部门经理→总经理 投标立项
标书申请 投标申请通过后，申请标书编制资源 商务→投标经理 投标立项
授权申请 需要厂家授权函时发起 商务→综合管理 投标立项
