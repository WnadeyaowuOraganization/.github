🔄 关键状态流转

线索状态机（leads.status） + 商机状态机（opportunities.stage，对齐 Salesforce 7 阶段）
📋 线索状态机 · leads.status
状态 触发条件 下一状态 典型动作
🆕 新线索 入口首次创建 培育中 / 已丢弃 业务员初判
🌱 培育中 开始跟进 SQL合格 / 休眠 发内容 / 邀约
✅ SQL合格 评分≥75 + 双向沟通 已转化 触发升级提醒
🔄 已转化 成功升级为商机 — 创建 opportunity
😴 休眠 30天无动作 重新激活 / 丢弃 进入培育池
❌ 已丢弃 不匹配 ICP — 归档
💰 商机状态机 · opportunities.stage

Closed Won 100% Closed 签约 + 首款到账
Closed Lost 0% Closed 丢单 + 复盘完成

📖 术语统一对照表

消除团队对「线索 / 询盘 / 矿场项目 / 商机」的概念混淆
当前平台用词 统一后定义 说明
线索（#36 外展） ✅ Lead 保持原义，统一落到 leads 表
商机（CRM） ✅ Opportunity 保持原义，落到 opportunities 表
询盘（经销 / 国贸） ⚠️ Early Opportunity 询盘 = 已过 Gate 的早期商机，直接建 opportunity，不走 lead（已有明确采购意向）
矿场项目 ⚠️ Lead-like + Manual Gate 矿场项目本质是线索，但因 B2G 决策周期长、不确定性大，由销售手动转换为商机
MQL / SQL ✅ 新增概念 MQL=评分 ≥50（市场合格）；SQL=评分 ≥75（销售合格，可升级）
