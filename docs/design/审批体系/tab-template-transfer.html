需求说明 模板迁移
▼
📌 页面目的 支持审批模板跨环境导入导出（含Schema+流程绑定配置），解决开发→生产环境模板同步以及新建模板配置的效率问题
👤 主要用户 吴耀（超管）, 吴延芝（管理员）
🎯 核心功能 批量导出模板（JSON Bundle） / 导入冲突检测（覆盖/重命名/跳过三选一） / 3步新建配置向导（分类→字段→审批流）
⚡ 关键操作 勾选模板导出JSON / 拖拽上传JSON导入 / 选择冲突处理策略 / 逐步完成新建向导
✅ 验收要点 导入前必须先执行冲突检测不允许静默覆盖 / 向导支持上一步/保存草稿 / 导出文件含完整Schema和流程绑定
