需求说明 询盘工作台
▼
📌 页面目的 管理商机前置阶段的询盘，支持报价和一键转商机（预填字段），对接直销/经销/国际三线的客户主动询盘入口。
👤 主要用户 商务专员，商务经理
🎯 核心功能 5状态Tab（全部/待报价/已报价/已转商机/已关闭） / 询盘主表格 / 转商机弹窗（预填客户名/产品需求/预算） / 报价与关闭操作
⚡ 关键操作 点击[报价]填写报价金额+有效期 / 点击[转商机]弹窗预填后创建商机（默认阶段C） / 点击[关闭]填写关闭原因
✅ 验收要点 状态Tab切换过滤结果正确 / 转商机弹窗预填客户名/产品需求/预算三字段 / 转商机成功后询盘状态变已转商机
