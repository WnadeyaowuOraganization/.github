需求说明 动态表单渲染器
▼
📌 页面目的 基于JSON Schema动态渲染审批表单，支持条件显隐、必填校验、文件上传等，让一套引擎驱动所有业务审批单
👤 主要用户 全员（发起审批时使用）
🎯 核心功能 12种字段类型渲染（文本/数字/金额/日期/用户选择器等） / 条件显隐引擎 / 必填红星标注 / 服务端+客户端双重校验
⚡ 关键操作 填写表单字段 / 上传附件 / 选择审批人 / 提交前触发本地校验再提交
✅ 验收要点 条件显隐纯前端计算无延迟 / readOnly字段灰色背景不可编辑 / autoGenerate字段禁止手动修改 / 驳回时意见必填
📖 详细设计文档：审批体系 详细设计.md

动态表单渲染器 — 根据 JSON Schema 自动生成表单 UI。本页展示渲染效果，实际由 #3168 组件实现。

字段类型支持

字段类型 Schema type 渲染组件 示例
单行文本 string a-input 项目名称
多行文本 string + textarea a-textarea 备注说明
数字 number a-input-number 报价金额
下拉选择 string + enum a-select 报价币种
单选 string + radio a-radio-group 报价类型
多选 array + checkbox a-checkbox-group 涉及产品线
日期 string + date a-date-picker 报价日期
文件上传 file a-upload 附件
用户选择器 user 自定义 审批人
部门选择器 department 自定义 所属部门
金额 money a-input-number + 前缀 ¥ 合同金额
