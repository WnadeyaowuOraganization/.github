### §1.3 业务背景

样品管理中心起源于 **D26 需求**（D80 后全新业务域），旨在打通以下全流程：

```
D3 设计出图
  ↓
自动解析色卡 + 材料清单（库存匹配）
  ↓
样品申请单生成（含缺量工单）
  ↓
支持中心审批
  ↓
工厂制作工单下发
  ↓
样品入库（多仓库）
  ↓
样品箱组配 / 借出给业务员
  ↓
现场演示 → 客户确认色卡
  ↓
归还 / 报损 / 归档
```

现状痛点：犀牛手动配色 + 纸质单据 + 手工 Excel，样品周转无迹可循，客户确认周期长。

### §1.4 主要用户角色

| 角色 | 核心使用场景 |
|------|------------|
| 设计师 | D3 一键生成申请单、在线配色、选用色卡/材料 |
| 设计主管 | 审批非标色卡/材料申请、审批配色方案 |
| 商务/销售 | 查看样品状态、发起借用申请 |
| 工厂 PMC | 接收制作工单、更新制作进度 |
| 仓管 | 多仓库库存管理、调拨登记、领用归还验收 |
| 业务员 | 外出携箱、H5 端借还登记 |

#### §2.1.3 D3 一键生成弹窗

**组件**：`D3GenerateDialog.vue`（a-modal，宽 720px）

**弹窗触发**：点击表格区右上角「D3 一键生成」按钮

**弹窗内容**：

1. **项目选择**：下拉读取 `design_project_selections` 表，只展示「已出图」项目
2. **读取来源提示**：自动显示 D3 项目色卡 + 材料选型清单条目数
3. **库存匹配结果表**：

| 列 | 说明 |
|----|------|
| 样品编码 | D3 出图自动识别 |
| 项目需要 | D3 清单数量 |
| 库存可用 | 实时查询 `stock` 表 |
| 缺量 | 需要 - 可用（≤0 显示灰色 0） |
| 制作工单 | 缺量 > 0 则显示「待生成 PRD-NEW-xx」 |

4. **底部说明**：「提交后将自动生成 N 张申请单 + M 张制作工单」
5. **按钮**：取消 · 提交并生成申请单

**数据源**：`POST /sample/request/d3-generate`  
**关联 Issue**：#2335（D3 集成解析）、#2336（库存匹配逻辑）、#2337（自动工单生成）、#2338（申请单提交）

**交互**：
1. 选择项目后调用 `GET /sample/request/d3-parse?project_id={id}` 获取清单
2. 清单获取后调用库存接口批量比对
3. 点击「提交并生成申请单」调用 `POST /sample/request/d3-generate`
4. 成功后关闭弹窗，刷新主表格，Toast 提示结果

**代码现状**：🔴 需新建  
**已有文件**：`tab-lifecycle.html` 区域3（行 274–352）

**三级边界**：
- ✅ Always：弹窗打开时必须加载库存匹配数据才可提交；缺量项必须高亮
- ⚠️ Ask First：是否支持手动调整「项目需要」数量后重新匹配
- 🚫 Never：不允许跳过库存匹配直接提交；不允许对未出图项目触发 D3 解析

### §6.3 H5 路由结构

```
/h5/sample                  → 首页（我的申请单 + 我持有的箱）

/h5/sample/kit/borrow       → 领用登记表单
/h5/sample/kit/return       → 归还登记表单（扫码入口）
/h5/sample/approval/:id     → 审批操作页（从企微消息链接跳入）
```

### §6.4 H5 UI 规范

- 最小点击区域：44×44px
- 底部固定操作栏（不悬浮）
- 列表卡片替代表格
- 扫码借还：调用 `uni.scanCode` 或 `wx.scanQRCode`

## §7 PageGuide 内容

### §7.1 这是什么

**样品管理中心** = D3 设计出图 → 库存匹配 → 申请审批 → 工厂制作 → 入库 → 借还全链路看板。

一站式整合了 5 个以前独立的纸质 / Excel 流程：
1. 样品申请与审批（样品全生命周期 Tab）
2. 色卡与材料标准化管理（色卡材料标准化 Tab）
3. 在线配色与方案确认（色卡配色器 Tab）
4. 跨仓库存可视化与调拨（多仓库库存联通 Tab）
5. 标准样品箱领用归还（样品箱制度 Tab）

### §7.2 解决什么问题

| 之前痛点 | 现在解法 |
|----------|----------|
| 犀牛手动配色，效果图来回发邮件 | 在线配色器实时预览 + 一键生成总部需求表 |
| 纸质样品申请单，审批周期长 | 线上审批流 + 审批状态实时可见 |
| 库存不知在哪个仓，找样品靠电话 | 多仓库存矩阵 + 仓库切换 |
| 样品箱借出去不知道在哪 | 箱追踪 + 超期企微自动推送 |
| 设计师随意用色，采购成本高 | 非标色卡/材料必须走主管审批 |

### §7.3 快速上手（3 步）

**第 1 步：D3 一键生成申请单**
> 在「样品全生命周期」Tab，点击右上角「D3 一键生成」，选择已出图项目，系统自动读取色卡清单并与库存对比，确认后一键生成申请单 + 工厂工单。

**第 2 步：支持中心审批**
> 主管在申请单列表点击「审批」，填写意见并提交，审批通过后工厂工单自动下发 PMC。

**第 3 步：借还登记**
> 业务员领用样品箱前，在「样品箱制度」Tab 填写领用登记（或手机 H5 扫码），系统记录持有人和计划归还日期；归还时仓管验收清点后确认归还，箱状态自动更新。
