📝 原型说明

# Section 功能描述 关键交互 / 数据来源
1 本周概要 4个核心KPI（活动总数 / 新增商机 / 签约项目 / 在跟管道值）+ AI生成一句话总结，吴耀10秒内把握本周整体态势 数据源：活动记录库、商机库、项目库；总结由 Qwen 生成
2 异常预警 4类预警（零活动 / 超7天未跟进 / Nudge未响应 / 新增丢标），灰阶+emoji区分等级，每条支持"立即处理/批量Nudge/查看明细/复盘"操作 规则引擎自动扫描，结果推送 Nudge 中心
3 各商务一句话摘要 6-8个商务卡片网格，每人一句话状态，异常成员自动高亮；点击进入商务详情页 Qwen 基于个人活动 + 项目汇总生成
4 管道健康度 圆环覆盖率仪表盘 + 5阶段堆叠条形图 + 阶段流转推进/退回健康度 管道库实时聚合；灰阶单色梯度可视化
5 推荐行动 AI 生成 3-5 条建议（标题 + 理由 + 动作），支持「采纳」一键落地到 Nudge 或日程 本地 Qwen 模型综合分析，含异常 + 管道 + 历史对比
· 页面定位：吴耀每周五下午查看，企微推送富文本卡片→点此查看完整周报；同时支持历史周报查看
· 自动生成：每周五16:00 触发定时任务，生成本周数据 + AI摘要 + 异常扫描 + 建议
· 推送通道：企微机器人推送富文本摘要卡片，底部附本页深链接
· 存档规则：所有周次永久保留，顶部下拉查看历史
