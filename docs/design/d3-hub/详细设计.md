## 1. 页面概述

- **路由**: `/wande/project-hub/d3` 或复用 `/wande/support/d3`（代码路径 `frontend/apps/web-antd/src/views/support/d3/`）
- **菜单层级**: 项目中心 > D3 参数化设计中心

- **所属模块**: `module:fullstack`（前后端联动，GH 插件为独立仓库 `wande-gh-plugins`）
- **对标矩阵**: Tacton CPQ（参数化配置）+ Configura CET（行业CAD工作流）+ Rhino+Grasshopper 原生（参数化引擎）+ SketchFab（3D 预览）
- **关键业务伙伴**:
  - PLM产品技术中心（#4024 Master）—— D3 BOM 输出为 PLM 零件主数据输入，PLM `PlmD3BridgeController` 已实现 `/api/plm/parts/sync-from-d3-library`

  - 提案引擎（`ProposalD3BridgeController`）—— 提案 → 自动创建 D3 设计会话
  - 超级员工矩阵（`D3TaskController`）—— D3 任务统计监控

### 新增表（#2268 + #1902 + #1922）

```sql
-- #2268 标注模板
CREATE TABLE wdpp_d3_annotation_template (
  id BIGINT PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  description VARCHAR(500),
  component_type_map JSON,  -- {"立柱": {...MaterialConfig}, "滑筒": {...}}
  project_id BIGINT NULL,   -- NULL=全局模板，有值=项目专属
  created_by BIGINT,
  created_at DATETIME,
  updated_at DATETIME
);

-- #2268 项目配色方案
CREATE TABLE wdpp_d3_color_scheme (
  id BIGINT PRIMARY KEY,
  project_id BIGINT NOT NULL,
  name VARCHAR(100) NOT NULL,  -- 标准版/高端版/经济版
  description VARCHAR(500),
  material_config JSON,
  is_active TINYINT DEFAULT 0, -- 当前激活方案
  created_at DATETIME,
  updated_at DATETIME
);

-- #1902 历史项目参数结构化
CREATE TABLE wdpp_d3_history_case (
  id BIGINT PRIMARY KEY,
  source_code VARCHAR(50),     -- 02993559/02993562/...
  project_name VARCHAR(200),   -- 丛林城堡乐园/儿童组合塔/...
  s3_path VARCHAR(500),
  -- 核心参数
  total_length_mm INT,
  total_width_mm INT,
  total_height_mm INT,
  platform_count INT,
  platform_heights JSON,       -- [1200, 2400, 3600]
  slide_types JSON,            -- ["直筒R800", "螺旋H4000"]
  main_material VARCHAR(100),
  estimated_weight_kg DECIMAL(10,2),
  estimated_volume_m3 DECIMAL(10,2),
  theme_tags JSON,             -- ["丛林", "城堡", "儿童"]
  customer_type VARCHAR(50),   -- 社区/学校/公园
  bom_snapshot JSON,
  render_images JSON,          -- S3 缩略图 URL 列表
  extracted_at DATETIME,
  created_at DATETIME
);

-- #1922 AI 匹配缓存（可选，提升性能）
CREATE TABLE wdpp_d3_ai_match_cache (
  id BIGINT PRIMARY KEY,
  feature_hash VARCHAR(64) UNIQUE,
  input_features JSON,
  top5_case_ids JSON,
  similarity_scores JSON,
  created_at DATETIME,
  expires_at DATETIME
);
```

## 7. PageGuide 内容

- **这是什么**：D3 参数化设计中心 — 基于 Rhino+Grasshopper 的游乐设备参数化设计平台，支持从提案到 BOM/图纸/3D 模型的全流程输出。
- **解决什么问题**：设计师手工画图效率低、参数变更牵一发动全身、历史项目经验难以复用；本中心通过参数化设计 + 构件库 + AI 案例匹配大幅降低定制件设计成本。
- **快速上手**：① Tab1 新建设计会话或从提案导入 → ② Tab4 AI 匹配相似历史案例作为基线 → ③ 参数调整+构件选择+材质标注 → ④ Tab5 渲染效果图 → ⑤ Tab7 导出 BOM 同步 PLM。
