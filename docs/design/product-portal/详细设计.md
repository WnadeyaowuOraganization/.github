## 1. 页面概述

- **路由**: `/wande/product-portal`（产品目录）、`/wande/product-portal/:id`（产品详情）、`/wande/product-portal/parts/:id`（备件目录）
- **菜单层级**: 产品中心 > 产品展示门户
- **权限标识**: `product-portal:view`（访客）、`product-portal:dealer`（经销商）
- **所属模块**: `module:fullstack`
- **新增业务标签**: `biz:product-portal`

### 权限模型（2级）

| 级别 | 角色 | 可见内容 |
|------|------|---------|
| L0 访客 | 未登录用户 | 产品目录浏览、基础参数、3D渲染图轮播、产品说明PDF下载 |
| L1 经销商 | 已登录经销商 | L0全部 + 3D爆炸图交互(Partful)、备件查找+下单、完整技术图纸(DWG/STEP/FBX)、尺寸测量、安装手册、价格信息 |

## 2. 页面布局

### 2.1 产品目录页 — `/wande/product-portal`

#### [筛选栏]
- **组件**: `a-select`(产品系列) + `a-select`(年龄段) + `a-input-search`(关键词)
- **数据源**: `GET /api/product-portal/products?series=&ageGroup=&keyword=&page=&size=`
- **交互**: 筛选条件变更后自动刷新列表，URL同步查询参数

#### [产品卡片网格]
- **组件**: `a-row` + `a-col`(span=8) + 自定义 `ProductCard.vue`
- **卡片内容**: 3D渲染缩略图(S3 presigned URL) / 产品名称+编码 / 年龄段+安全区域+坠落高度 / 认证标识(EN 1176/TÜV) / 锁定标识(未登录时)
- **交互**: 点击卡片 → 跳转 `/wande/product-portal/:id`

### 2.2 产品详情页 — `/wande/product-portal/:id`（核心页面）

#### 左侧面板（40%宽度）

##### [产品参数表]
- **组件**: `a-descriptions`
- **数据源**: `GET /api/product-portal/products/:id`
- **字段**: 产品编码、适用年龄、安全区域面积、坠落高度、认证标准、活动类型

##### [技能雷达图]
- **组件**: ECharts radar chart
- **数据源**: 同上，`skillChart`字段

##### [询价操作]
- **组件**: `a-button`(primary)
- **交互**: 
  - L0: 「登录后查看价格」→ 跳转登录
  - L1: 「加入询价单」→ 添加到询价车

#### 右侧面板（60%宽度）

##### [3D查看器] — 核心区域
- **组件**: Partful SDK iframe嵌入 / 降级为3D渲染图轮播(L0)
- **实现方式**:
  ```html
  <!-- L1经销商: Partful SDK嵌入 -->
  <iframe 
    src="https://app.partful.io/embed/{partful_product_id}?token={dealer_token}" 
    width="100%" height="400" 
    frameborder="0" 
    allow="fullscreen"
  ></iframe>
  
  <!-- L0访客: 3D渲染图轮播降级 -->
  <a-carousel>
    <img v-for="img in product.renderImages" :src="img.url" />
  </a-carousel>
  ```
- **Partful SDK功能映射**:

| 工具 | Partful功能 | 权限 |
|------|------------|------|
| 360°旋转 | 内置3D旋转 | L0(用轮播降级) / L1(Partful原生) |
| 爆炸图拆解 | Partful Exploded View | L1 |
| 尺寸测量 | Partful Measurement Tool | L1 |
| 备件查找 | Partful Parts Selection + Click-to-order | L1 |
| 安全区域可视化 | 自定义叠加层(overlay on Partful) | L1 |

##### [3D工具栏]
- **组件**: 自定义 `ToolBar.vue`
- **布局**: 底部横排5个按钮
- **交互**: 
  - L0: 仅360°旋转可用，其余显示🔒图标+点击弹出登录引导
  - L1: 全部可用，点击切换Partful SDK对应功能模式

##### [资料下载区]
- **组件**: `a-list` + 自定义 `DownloadItem.vue`
- **数据源**: `GET /api/product-portal/products/:id/documents`
- **权限控制**:

| 文件类型 | 权限 | S3路径模式 |
|---------|------|-----------|
| 产品说明书.pdf | L0(公开) | `wande-nas-sync/产品资料/{code}/说明书.pdf` |
| 平面图.dwg | L1(经销商) | `wande-nas-sync/产品资料/{code}/views.dwg` |
| 安全区域图.dwg | L1(经销商) | `wande-nas-sync/产品资料/{code}/safety-area.dwg` |
| 3D模型.fbx | L1(经销商) | `wande-nas-sync/06 定制化产品标准化模型库/{code}.fbx` |
| 3D模型.step | L1(经销商) | `wande-nas-sync/06 定制化产品标准化模型库/{code}.step` |

- **下载实现**: 后端生成S3 presigned URL → 前端直接下载（不经过后端中转，降低带宽成本）

## 3. API契约

### GET /api/product-portal/products
产品列表（公开）
- **参数**: `series`(string), `ageGroup`(string), `keyword`(string), `page`(int), `size`(int)
- **返回**: `{ total, list: [{ id, code, name, series, ageGroup, safetyArea, fallHeight, certifications[], thumbnailUrl, hasPartfulModel }] }`

### GET /api/product-portal/products/:id
产品详情（公开基础 + 经销商增强）
- **参数**: `id`(long)
- **返回**: `{ id, code, name, series, description, ageGroup, safetyArea, fallHeight, certifications[], activities[], skillChart{}, renderImages[], partfulProductId(仅L1), priceRange(仅L1) }`

### GET /api/product-portal/products/:id/documents
产品资料列表（权限过滤）
- **参数**: `id`(long)
- **返回**: `{ documents: [{ id, name, type(pdf/dwg/fbx/step), size, accessLevel(L0/L1), downloadUrl(presigned) }] }`
- **权限**: L0用户只返回`accessLevel=L0`的文档，L1返回全部

### GET /api/product-portal/products/:id/parts
产品备件列表（经销商专属）
- **参数**: `id`(long), `keyword`(string)
- **返回**: `{ parts: [{ partCode, partName, material, specification, stockStatus, unitPrice, partfulPartId }] }`
- **权限**: 仅L1经销商可访问

### POST /api/product-portal/inquiry
加入询价单
- **参数**: `{ productId, quantity, note }`
- **权限**: 仅L1经销商

### POST /api/product-portal/parts/cart
备件加入购物车
- **参数**: `{ partCode, productId, quantity }`
- **权限**: 仅L1经销商

## 4. 数据库设计

### 已有表（复用）
- `sys_user` — 用户表（RBAC权限判断）
- `wande_product` — 产品基础表（如有，需确认字段覆盖度）

### 新增表

#### product_portal_product（产品展示信息表）
```sql
CREATE TABLE product_portal_product (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    product_code VARCHAR(50) NOT NULL COMMENT '产品编码 如WD-OC-301',
    product_name VARCHAR(200) NOT NULL COMMENT '产品名称',
    series VARCHAR(50) COMMENT '产品系列: slide/climb/swing/sand/fitness/theme/1165',
    age_group VARCHAR(20) COMMENT '适用年龄: 0-3/3-6/6-12/12+',
    safety_area DECIMAL(10,2) COMMENT '安全区域面积(m²)',
    fall_height DECIMAL(5,2) COMMENT '坠落高度(m)',
    certifications JSON COMMENT '认证列表 ["EN 1176","TÜV"]',
    activities JSON COMMENT '活动类型列表',
    skill_chart JSON COMMENT '技能雷达图数据',
    description TEXT COMMENT '产品描述',
    partful_product_id VARCHAR(100) COMMENT 'Partful平台产品ID（L1功能）',
    price_range VARCHAR(50) COMMENT '价格区间（L1可见）',
    thumbnail_s3_key VARCHAR(500) COMMENT 'S3缩略图路径',
    render_images JSON COMMENT '3D渲染图S3路径列表',
    sort_order INT DEFAULT 0,
    status TINYINT DEFAULT 1 COMMENT '1=上架 0=下架',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    UNIQUE KEY uk_product_code (product_code)
) COMMENT='产品展示门户-产品信息表';
```

#### product_portal_document（产品资料表）
```sql
CREATE TABLE product_portal_document (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    product_id BIGINT NOT NULL COMMENT '关联产品ID',
    doc_name VARCHAR(200) NOT NULL COMMENT '文件名称',
    doc_type VARCHAR(20) NOT NULL COMMENT '文件类型: pdf/dwg/fbx/step',
    s3_key VARCHAR(500) NOT NULL COMMENT 'S3文件路径',
    file_size BIGINT COMMENT '文件大小(bytes)',
    access_level VARCHAR(10) DEFAULT 'L1' COMMENT '访问级别: L0=公开, L1=经销商',
    sort_order INT DEFAULT 0,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    KEY idx_product_id (product_id)
) COMMENT='产品展示门户-资料文件表';
```

#### product_portal_part（产品备件表）
```sql
CREATE TABLE product_portal_part (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    product_id BIGINT NOT NULL COMMENT '关联产品ID',
    part_code VARCHAR(50) NOT NULL COMMENT '零件编号',
    part_name VARCHAR(200) NOT NULL COMMENT '零件名称',
    material VARCHAR(100) COMMENT '材质',
    specification VARCHAR(200) COMMENT '规格参数',
    stock_status VARCHAR(20) DEFAULT 'in_stock' COMMENT '库存状态: in_stock/low/out_of_stock',
    unit_price DECIMAL(12,2) COMMENT '单价',
    partful_part_id VARCHAR(100) COMMENT 'Partful零件ID（用于3D高亮联动）',
    sort_order INT DEFAULT 0,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    KEY idx_product_id (product_id),
    UNIQUE KEY uk_part_code (part_code)
) COMMENT='产品展示门户-备件表';
```

| 零件高亮 | ✅ 点击+装配树 | ✅ 支持 | ❌ 不支持 |
| STEP解析 | ✅ CLI(cadquery) | ✅ 云端 | ✅ 支持 |

| 测量工具 | ✅ 内置 | ✅ 内置 | ✅ 内置 |
| 导出 | STL/OBJ/glTF/PLY/PNG/GIF | 有限 | 有限 |

来源: [ExplodeView GitHub](https://github.com/vvlars-cmd/explodeview) / [explodeview.com](https://explodeview.com)

### 6.2 技术架构

```
                    万德平台(wande-play)
                         │
      ┌──────────────────┼──────────────────┐
      │                  │                  │
  STEP模型(S3)     ExplodeView           前端Vue组件
  227个.step文件    Docker服务栈          PartfulViewer.vue
      │                  │                  │
      ▼                  ▼                  ▼
  CLI批量处理      FastAPI STEP转换器    CDN嵌入查看器
  python3           :8787               unpkg.com/explodeview
  explodeview-process.py                     │
      │                                      ▼
      ▼                              浏览器WebGL渲染
  processed assets/                  (Three.js r170)
  ├─ parts/*.stl
  ├─ manifest.json
  ├─ assemblies.json
  └─ config.json
      │
      ▼
  上传到S3
  wande-nas-sync/product-portal-assets/{code}/
```

### 6.3 接入流程

```
Step 1: G7e部署ExplodeView Docker服务栈
  docker-compose up → :8080(查看器) + :8787(STEP转换器) + :3000(cycleCAD)

Step 2: 批量处理STEP模型（G7e Pipeline）
  for each .step in S3/06 定制化产品标准化模型库/:
    python3 bin/explodeview-process.py <file.step> output/ \
      --name "海洋主题组合滑梯" --brand "WanDe" --tolerance 0.5
  → 生成 parts/*.stl + manifest.json + assemblies.json + config.json

Step 3: 上传processed assets到S3
  aws s3 sync output/ s3://wande-nas-sync/product-portal-assets/{code}/

Step 4: 前端嵌入
  <div id="viewer" data-stp-viewer="/api/product-portal/assets/{code}/"
       data-brand="WanDe" data-product-name="{name}"
       style="width:100%; height:500px">
  </div>
  <script src="https://unpkg.com/explodeview"></script>

Step 5: 后端API提供S3 presigned URL给assets目录
  GET /api/product-portal/assets/{code}/ → presigned URLs for parts/*.stl等
```

## 8. PageGuide内容
- **这是什么**: 产品展示门户是万德平台的对外产品展示窗口，经销商可以360度查看产品3D模型、浏览爆炸图拆解结构、查找和订购备件。
- **解决什么问题**: 替代传统的PDF产品目录和电话/邮件沟通备件需求，让经销商自助完成产品了解和备件订购。
- **快速上手**: 
  1. 浏览产品目录，点击感兴趣的产品查看详情
  2. 使用3D查看器旋转、拆解产品，查看内部结构
  3. 在备件中心点击零件直接加入购物车
