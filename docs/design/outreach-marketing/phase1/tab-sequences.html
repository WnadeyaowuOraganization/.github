🔌 核心 API 端点
GET /api/v1/sequences
列出全部序列，支持 status/channel/owner 筛选
POST /api/v1/sequences
创建新序列，body 接收完整 steps 配置
GET /api/v1/sequences/{id}
获取单条序列详情 + metrics
PATCH /api/v1/sequences/{id}/status
暂停 / 恢复 / 归档
POST /api/v1/sequences/{id}/enroll
批量入组线索（接收 lead_ids 数组）
GET /api/v1/sequences/{id}/metrics
获取序列实时指标
