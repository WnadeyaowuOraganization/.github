需求说明 流程监控
▼
📌 页面目的 汇聚该商机下所有审批流状态（报价/合同/投标审批），商务一屏看清哪个流程卡在谁手里，配合协同模块催办。
👤 主要用户 商务专员，商务经理
🎯 核心功能 审批流列表（流程类型/当前节点/处理人/发起时间/等待天数） / 审批状态（待提交/审批中/已通过/已驳回） / 催办入口（等待天数>=5天标红）
⚡ 关键操作 查看各审批流当前节点 / 点击[催办]向处理人发送提醒 / 点击[详情]查看完整审批历史
✅ 验收要点 审批流列表按发起时间排序 / 等待>=5天的条目红色标注 / 催办操作调用notification/remind API
📖 详细设计文档：CRM商务中心 详细设计.md（商机详情相关）
