需求说明 投标申请
▼
📌 页面目的 管理直销业务线的投标申请全流程（新建→销售主管审批→中标/未中标结果录入），对接招标公告来源的商机，避免投标决策随意性。
👤 主要用户 商务专员（申请），销售主管（审批）
🎯 核心功能 5状态Tab（草稿/待审批/审批中/已通过/未通过） / 投标列表（截止日期临近变橙/过期变红） / 新建投标申请弹窗（6字段含投标策略） / 审批流转+结果录入
⚡ 关键操作 点击[提交审批]发起审批流 / 点击[上传标书]附件上传 / 审批通过后录入投标结果（中标/未中标）
