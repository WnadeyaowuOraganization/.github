# D3参数化设计 v2.0 - Sprint 2026-03-28 排程计划

## 模块目标
电池包体系 + AI集成 + Web平台

## 架构分层
1. **Core层**: 共享模块、云端API、产品库、标准库
2. **电池包层**: 几何生成组件（立柱/平台/楼梯/荡桥/滑桶等）
3. **AI层**: 知识库、智能推荐、主动建议、合规报告
4. **Web层**: 参数配置器、电池包目录、BOM管理、3D预览
5. **生产链路**: 排料、标注、图纸输出、STEP导出

## 关键里程碑

1. **M1 (Week 2)**: Core层完成，电池包开发环境就绪
2. **M2 (Week 4)**: 核心电池包完成，可生成基础游乐设施
3. **M3 (Week 6)**: Web平台基础功能可用，设计师可上传和配置
4. **M4 (Week 8)**: AI知识库就绪，可提供智能建议
5. **M5 (Week 10)**: 生产链路打通，可输出加工图纸
6. **M6 (Week 12)**: 端到端验证完成，全流程跑通

## 风险点

1. **#1920 仓库重构** — 影响所有插件开发，必须最先完成
2. **#1909 核心电池包** — 是整个D3体系的基础，延期将连锁影响
3. **胡总确认事项** — #1903 选型规则、#1904 模具标准需要确认
4. **AI知识库质量** — #1867 知识体系构建质量决定AI功能效果
5. **CadQuery环境** — #1855 G7e环境配置可能影响AI Agent开发
