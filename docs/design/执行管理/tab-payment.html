📌 页面目的 管理各期回款计划与资料Checklist，解决「回款资料分散、催款无依据」痛点，资料完备度直接触发回款条件，支持一键催交企微通知
👤 主要用户 财务, 商务负责人, 项目经理
🎯 核心功能 回款进度三卡片（合同总额/已回款/待回款） / 回款计划表（含资料状态N/M） / 各期资料Checklist（D68体系） / 催收管理+自动提醒 / 企业信息库 / 资料需求模板库
⚡ 关键操作 查看各期资料完成度 / 点击催交触发企微通知 / 登记实际回款金额和日期 / 选套用资料模板
✅ 验收要点 金额后端计算不在前端汇总（防精度丢失） / Checklist进度条N/M显示正确 / 催交后催收记录自动新增并同步CRM活动日志 / 各期资料状态与回款条件关联
