## 2. 页面布局 — 4大分组·18个Tab

### 2.1 分组1: 平台系统（9个Tab）

#### Tab 1: 总览仪表盘（默认页）
- **组件**: KPI卡片(a-statistic) + 服务状态网格 + 告警列表(a-list)
- **数据源**: `GET /wande/cockpit/dashboard/stats`（聚合接口）
- **KPI**: 服务健康度 / 活跃CC数(8/20) / 今日部署次数 / Token Pool余额
- **服务状态**: 6个服务（后端:6040 / 前端:8083 / CI后端:6041 / CI前端:8084 / PG:5433 / Redis:6380）
- **告警**: 最近10条告警，按时间倒序，左侧色条标注级别

#### Tab 2: Claude Office
- **组件**: CC状态矩阵(grid) + 研发经理日志(log-viewer) + Kanban看板(a-card)
- **数据源**: `GET /wande/cockpit/claude/sessions` + `GET /wande/cockpit/issues/kanban`
- **CC矩阵**: kimi1~kimi20网格，每个格子显示：目录/Issue号/状态(🟢/🔴/🟡)
- **Kanban**: 5列（Plan/Todo/In Progress/E2E Fail/Done），显示计数+最新3条

#### Tab 3: Token Pool管理
- **组件**: 表格(a-table) + 告警规则(a-form)
- **数据源**: `GET /system/dashboard/token-pool/status` + `GET /wande/token-pool/alert/rules`
- **已有Controller**: DashboardTokenPoolController + TokenPoolAlertController

#### Tab 4: 采集管控面板
- **组件**: 管线状态表(a-table) + 质量雷达(自定义) + 异常面板(a-alert)
- **数据源**: `GET /wande/pipeline/funnel` + `GET /wande/cockpit/data-quality`
- **关联Issue**: #3300~#3304(管线监控) + #2925/#2926(数据质量)

#### Tab 5: 定时任务管理
- **组件**: Cron表格(a-table) + 操作按钮
- **数据源**: `GET /wande/dashboard/cron/list`
- **已有Controller**: CronTaskManagementController + DashboardCronOperationController

#### Tab 6: 实时日志流（#3481 新增）
- **组件**: 过滤器(a-select+a-input) + 日志列表(虚拟滚动) + 来源统计(a-progress)
- **数据源**: `GET /wande/cockpit/logs/stream`（SSE或WebSocket）
- **日志类型**: CC会话 / CI流水线 / API请求 / 部署
- **交互**: 实时追加+暂停按钮，关键词高亮

#### Tab 7: FinOps成本看板（#3482 新增）
- **组件**: 成本卡片(a-statistic) + 预算对比条(a-progress) + 明细表(a-table)
- **数据源**: `GET /wande/cockpit/finops/summary` + `GET /wande/cockpit/finops/details`
- **成本分类**: AI模型(智谱/中转站/本地) / 云资源(EC2/Lightsail) / SaaS(GitHub/域名)

#### Tab 8: GPU资源监控
- **组件**: 利用率卡片 + vLLM队列状态
- **数据源**: `GET /api/monitor/gpu/status`
- **已有Controller**: GpuMonitorController

#### Tab 9: 外部工具管理
- **组件**: 工具表格(a-table) + 状态指示器
- **数据源**: `GET /monitor/ext-tool/list`
- **已有Controller**: ExtToolController系列(5个)

### 2.2 分组2: 开发者协同（5个Tab）

#### Tab 1: 确认中心
- **组件**: 确认卡片列表(a-card) + 确认/拒绝按钮
- **数据源**: `GET /api/dashboard/confirmations/pending`
- **已有Controller**: ConfirmationCenterController

#### Tab 2: Issue看板
- **组件**: Kanban 5列 + 筛选器
- **数据源**: `GET /wande/cockpit/issues` (GraphQL代理)
- **已有Controller**: DashboardIssueController + CfdController

#### Tab 3: Agent效率看板（#2849）
- **组件**: 效率表格(a-table) + 趋势图
- **数据源**: `GET /wande/cockpit/agent/efficiency`
- **已有Controller**: AgentEfficiencyController

#### Tab 4: 验收队列（#2851）
- **组件**: 待办列表(a-list) + 优先级标签 + 响应趋势
- **数据源**: `GET /wande/dashboard/acceptance/queue`
- **已有Controller**: DashboardAcceptanceQueueController

#### Tab 5: DORA指标（#2275）
- **组件**: 4个DORA卡片(a-statistic) + 错误滚动条
- **数据源**: `GET /api/dora/metrics`
- **已有Controller**: DoraMetricsController

### 2.3 分组3: 安全审计（2个Tab）

#### Tab 1: 安全审计日志（#3483/#2296）
- **组件**: 审计表格(a-table) + 异常检测面板 + 合规状态
- **数据源**: `GET /wande/cockpit/audit/logs` + `GET /wande/cockpit/audit/anomalies`
- **关联代码**: views/wande/audit-log/（1个Vue已有）

#### Tab 2: Prompt管理（#3484）
- **组件**: 依赖图(自定义tree) + 一致性报告(a-table) + 文件列表(a-table)
- **数据源**: `GET /wande/cockpit/prompts/list` + `GET /wande/cockpit/prompts/consistency`
- **交互**: 查看diff / 一键同步 / 编辑

### 2.4 分组4: 业务板块（2个Tab）

#### Tab 1: 问题发现（#2078）
- **组件**: 模式切换(a-radio-group) + 问题/机会列表(a-list)
- **数据源**: `GET /wande/problem-discovery/list` + `GET /api/dashboard/problems/diagnosis`
- **已有Controller**: ProblemDiscoveryController + ProblemDiagnosisController

#### Tab 2: AI对话监控（#2298）
- **组件**: KPI卡片 + 来源分布(a-progress) + 对话列表(a-table)
- **数据源**: `GET /wande/cockpit/chat/stats`
- **关联代码**: views/dashboard/cockpit/ai-chat-monitor/(5个Vue)
