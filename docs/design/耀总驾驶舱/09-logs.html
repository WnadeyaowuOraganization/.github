需求说明 日志流
▼
📌 页面目的 实时展示平台运行日志流，支持关键词过滤和日志级别筛选，是定位线上问题的第一现场
👤 主要用户 伟平（开发者）, 吴耀（超管）
🎯 核心功能 实时日志滚动 / 日志级别筛选（INFO/WARN/ERROR） / 关键词搜索 / 时间范围选择
⚡ 关键操作 输入关键词过滤日志 / 暂停/恢复实时滚动 / 导出日志片段
✅ 验收要点 日志实时推送延迟<5s / 过滤结果准确 / ERROR级别红色高亮
