需求说明 售后质保
▼
📌 页面目的 管理项目竣工验收（四方参与）、整改工单、质保期及售后服务工单，验收通过后自动触发质保计时+最终回款条件满足+设备档案自动生成三联动
👤 主要用户 项目经理, 售后工程师, 甲方（客户）
🎯 核心功能 质保状态卡片（状态/到期日/待处理工单） / 四方验收管理（甲方/工程队/监理/设计部门） / EN1176/GB19272检查项模板 / 整改工单管理 / 质保与维保计划 / 售后工单 / 设备档案
⚡ 关键操作 标记验收检查项通过/不通过 / 新增整改工单 / 登记售后服务工单 / 查看设备档案
✅ 验收要点 未通过整改的检查项「标记通过」按钮disabled / 验收全通过后质保期自动开始计时 / 质保期时长不硬编码（各合同可配置） / 设备档案验收后自动生成
