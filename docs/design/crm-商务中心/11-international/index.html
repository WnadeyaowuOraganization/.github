原型说明 — §11 国贸专属页（5 Tab 架构）

Tab Tab 名称 文件名 承接 Issue 功能描述
Tab 1 PI 形式发票 tab-pi.html #3104 从商机报价一键生成 PI，币种/条款/明细可编辑，支持中英双语/英文/俄文 PDF 导出，版本历史管理
Tab 2 发货管理 tab-shipping.html #3106 装箱单+报关单+海运/空运跟踪时间线，BL/AWB 号实时状态，地图定位船/飞机位置
Tab 3 出口单据 tab-docs.html #3107 CI/PL/BL/CO/FormE 5 类出口单据自动生成+归档，模板管理（公司抬头/签章），批量打包 ZIP
Tab 4 客户信用额度 tab-credit.html #3109 授信+在途应收+账龄分析+超额预警，使用率≥80% 黄预警，≥100% 红预警拦截报价（经销+国贸通用）
Tab 5 汇率管理 tab-fx.html #3111 实时汇率+报价锁汇+收款汇率+汇兑损益，USD/EUR/JPY/GBP vs CNY，30 天趋势图
🔗 架构说明
本页面采用多文件 Tab 架构，每个 Tab 独立 HTML 文件，通过 iframe 加载。修改单个 Tab 不影响其他 Tab 的迭代。 详细设计章节对应：§11.1 PI / §11.2 发货 / §11.3 单据 / §11.4 信用 / §11.5 汇率。
